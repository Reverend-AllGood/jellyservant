## Setup
1. Clone the repo
2. Place `jellyfin_skeleton_meal.jpg` in the root folder
3. Run `docker compose up -d --build`
4. Open `http://<your-server>:5000`
5. Go to the Config tab and enter your Jellyfin URL, API key, bridge domain and Nginx credentials

   Detailed instructions will be available in time for full 1.0 release.
